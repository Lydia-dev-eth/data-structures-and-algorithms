# Programming Assignments

This directory contains programming assignments completed as part of university courses and online learning platforms.

Unlike individual coding problems, these assignments are larger projects that often combine multiple algorithms, data structures, and software engineering concepts.

## Structure

```text
mit/
university/
other/
```

Each assignment directory may include:

* **README.md** – Assignment overview and objectives.
* **Source files** – Project implementation.
* **Test files** – Unit tests or provided testing scripts.
* **notes.md** – Personal observations and lessons learned.

## Purpose

The purpose of this directory is to document complete programming projects and demonstrate the practical application of algorithms and software development practices.
